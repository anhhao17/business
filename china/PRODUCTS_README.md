# Products CSV - How to Use

## File: `products.csv`

30 products sourced from multiple Chinese platforms with real supplier links.

## Columns

| Column | Description |
|---|---|
| ID | Product number |
| Product Name | Descriptive product name |
| Category | Tech Accessories / Smart Home / Wearables / Fashion / Beauty / Home |
| Source Platform | 1688 / Alibaba / Made-in-China / Factory / Vietnam Retail (reference) |
| Source Link | **Direct link to supplier/product page** |
| Source Price (local) | Price in original currency (CNY or USD) |
| Source Currency | CNY or USD |
| Source Price (USD) | Converted to USD (1 CNY ≈ $0.14) |
| MOQ | Minimum order quantity |
| Estimated Sell Price (VND) | Target retail price in Vietnam |
| Estimated Sell Price (USD) | Converted sell price (1 USD ≈ 25,000 VND) |
| Estimated Margin (%) | Gross margin after product cost (before ads/fees) |
| Shipping Weight (g) | Estimated unit weight (affects shipping cost) |
| Seasonality | Year-round or Summer |
| Regulation Risk | Low / Medium (cosmetics & batteries need checks) |
| Notes | Key specs, features, supplier details |
